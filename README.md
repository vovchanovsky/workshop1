# workshop1 

## Task 1
| Git Command | Explanation |
|------------|-------------|
| `git init` | Initializes a new local Git repository. |
| `git add <file>` | Stages a specific file to be included in the next commit. |
| `git commit -m "<message>"` | Creates a commit with the staged changes. |
| `git branch -M main` | Renames the current branch to **main**. |
| `git remote add origin <URL>` | Connects the local repository to a remote GitHub repo. |
| `git push -u origin main` | Pushes the main branch to GitHub and sets upstream tracking. |
| `git add .` | Stages all changed files. |
| `git pull` | Fetches and merges changes from the remote repository. |
| `git push` | Uploads local commits to GitHub. |
| `git commit --amend -m "<message>"` | Modifies the most recent commit. |
| `git commit -a -m "<message>"` | Stages and commits all tracked modified files. |

## Task 3
| Git Command | Explanation |
|------------|-------------|
| `git branch old` | Creates a new branch named **old**. |
| `git branch` | Lists existing branches. |
| `git checkout old` | Switches to the **old** branch. |
| `git commit -a -m "<message>"` | Stages and commits all tracked modified files. |
| `git pull` | Fetches and merges changes from the remote repository. |
| `git push -u origin old` | Pushes the **old** branch to GitHub and sets upstream tracking. |
| `git push` | Uploads local commits to GitHub. |
| `git checkout main` | Switches back to the **main** branch. |
